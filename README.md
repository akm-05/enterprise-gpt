# Enterprise GPT — AI Workspace Landing Page

A high-fidelity SaaS landing page inspired by modern AI workspace tools, built using Next.js, Tailwind CSS, GSAP, and Framer Motion.

This project focuses on pixel-accurate UI, smooth scroll-driven interactions, and clean component architecture.

---

## Overview

This project replicates a modern AI product landing page with an emphasis on:

- Visual precision (spacing, typography, layout)
- Scroll-based interactions and motion design
- Modular, reusable components
- Responsive behavior across devices

---

## Features

- Scroll-triggered animations using GSAP (ScrollTrigger)
- Smooth reveal transitions with Framer Motion
- Stacked card interaction (scroll-driven)
- Custom local font integration
- Fully responsive layout (desktop, tablet, mobile)
- Clean and scalable component structure

---

## Tech Stack

- Next.js (App Router)
- Tailwind CSS
- TypeScript
- GSAP (ScrollTrigger)
- Framer Motion

## Installation

Clone the repository:
```bash
git clone https://github.com/akm-05/enterprise-gpt
cd enterprise-gpt
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open in browser:
```bash
http://localhost:3000
```
---

## Build
```bash
npm run build
```

## Deployment

Recommended platform: Vercel

You can deploy directly via the Vercel dashboard or CLI.
